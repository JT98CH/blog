Ensign College Collection

While working for Ensign Collect, I could pull data from different databases using SQL on Snowflake.

But also I worked with CSV files, I analyzed data from Qualtrics surveys. I changed the data before publishing here because I'm aware of the sensitive/private data; that's why you're allowed to use only "Academic Challenge," "Executive Summary," "Background," and "Other."

I loved the idea of creating interactive dashboards because it helps to engage the insights. During my time at Ensign College, I led the implementation of Power BI in the department. That assignment, using Tableau dashboards to "translate" them to Power BI, was an excellent experience in increasing my data visualization and storytelling expertise.

Personal Collection

I built this dashboard to illustrate some data for a website project for my brother. The website's purpose was to show and track opiods data.

The Uvalde school shooting impacted me, so I did a little research. When I looked for data and visualizations about it, I didn't like them, so I decided to create my own dashboard. I discovered and illustrated the two main issues about this topic. I don't want to conclude, but apparently, "Legally obtained weapons" and "mental health issues" are relevant factors for this controversial issue.
